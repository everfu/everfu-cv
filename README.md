# everfu-cv

我自己的简历，支持打印。

## 技术栈

- Next.js 16
- React 19
- TypeScript
- Sass
- Iconify

## 本地开发

```bash
pnpm install
pnpm dev
```

默认访问地址：

```text
http://localhost:3000
```

如需让 Open Graph / canonical 指向正式站点，可设置：

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.example
```

## 构建

```bash
pnpm build
pnpm start
```
